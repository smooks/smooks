Smooks Templating Cartridge.

Overview

This Cartridge provides node-level templating capabilities within Smooks. This allows template based transformations (node-level) to be used in Smooks.

Supported templating frameworks:

  • FreeMarker
  • XSLT
  • StringTemplate

Using This Cartridge

See Milyn Tutorials for examples of using the templating solutions provided in this Cartridge.
